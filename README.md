Create an simple ecommerce system:

Admin site:
    Products management (CRUD)
    Orders management (Change order status)
    Client site:
    Login/Register

    Track order status

Suggestion tools:

    DBeaver: powerful DB management tool than SSMS
    Git remote: GitHub/GitLab
